What is a DOM and what is a purpose of DOM?

Ans. The Document Object Model (DOM) is the model which provide us way to interact with web pages. The main purpose of DOM is to make website dynamic so that we can interact with website.

How will you select HTML elements using DOM? Write the name of DOM methods for selecting HTML elements.

Ans. We can select HTML elements by tag name, id, class name, querySelector and querySelectorAll using DOM. Example:
1. getElementsByTagName('h1'),
2. getElementsByClassName('class-name'),
3. getElementById('id'),
4. querySelector('tag-name or #id or .class-name'),
5. querySelectorAll('tag-name or #id or .class-name div')

The name of DOM methods for selecting HTML elements are
1. getElementsByTagName(),
2. getElementsByClassName(),
3. getElementById(),
4. querySelector() and
5. querySelectorAll()

What is event bubbling?

Ans. The event bubbling is a concept in the DOM. When the event happen in the child html elements, the child html elements parse the event in his parent html elements. This processing known as event bubbling in DOM.
